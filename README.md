# MedSchedule Hub - Landing Page Odontológica

Este projeto foi desenvolvido como uma solução ágil de agendamento para clínicas, utilizando ferramentas de **No-Code e Low-Code (Lovable e Buildfy)**.

## 🛠️ Tecnologias e Ferramentas
* **Interface:** Desenvolvida com Lovable (React/Tailwind por baixo do capô).
* **Banco de Dados:** Estruturado com Supabase.
* **Prototipagem:** Buildfy.

## 🎯 Objetivo do Projeto
Criar uma experiência de usuário (UX) fluida para pacientes, permitindo o agendamento rápido de consultas e apresentação dos serviços da clínica.
